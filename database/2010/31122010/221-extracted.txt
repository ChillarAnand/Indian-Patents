(57) Abstract : 
A graphical user interface (GUT) can be presented on a remote control accessory device that has user input and display devices. The 
GUI can be defined and managed by a portable media device that is controlled using the GUL The portable media device can provide 
the accessory with a GUI image to be displayed. The accessory can send information to the portable media device indicative of a user 
operation of an input device in response to the displayed image. The portable media device can process this input to identify the action 
requested by me user and take the appropriate action, which can include updating the GUI image provided to the accessory.
