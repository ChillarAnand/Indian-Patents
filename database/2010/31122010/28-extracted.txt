(57) Abstract : 
A system and method for managing and controlling access to wireless local area network is provided. The system includes a wireless 
local area network (102), configured to be accessed by one or more guest data processing systems (108, 110 and 112) only after 
successful authentication. The system further includes, plurality of security keys (108a, 110a and 112a), wherein each of the security 
keys (108a, 1l0a and 112a) is configured to, store authentication data and facilitate extraction of the authentication data by any of the 
guest data processing systems (108, 110 and 112), with which the security key is engaged. Furthermore, the system includes an 
authentication device (106) configured to receive authentication data from each of the guest data processing systems (108, 110 and 
112), and at least, based on the authentication data communicated by the respective guest data processing systems (108, 110 and 112), 
allow or deny accessing the wireless local area network (102) by the respective guest data processing systems (108, 110 and 112).
