(57) Abstract : 
The invention relates to a lock comprising a housing (14, 55), a key operated lock cylinder (13, 51), which is arranged rotatable in the 
lock housing and a lock catch (11, 60), which is connected to the lock cylinder and rotatable between a locking position and a 
releasing position. The lock is arranged to be selectively set in a key operated mode, wherein the lock catch is rotatable with the lock 
cylinder, or in a push button operated mode, wherein the lock catch is movable by axial displacement of the lock cylinder.
