Abstract  =>  A method of automatically generating text messages in response to a telephone call is disclosed and may include receiving an incoming telephone call at a text message capable device from a calling device, determining whether a text message engine within the text message capable device is enabled, automatically generating a text message when the text message engine is enabled and transmitting the text message to the calling device.
