Abstract  =>  A method of operating a navigation system to provide a route guidance message for traveling a route is disclosed. A preferred name of a feature visible from a road segment is obtained in a native language from a geographic database associated with the navigation system. The parts-of-speech of the preferred name are identified and converted into a target language text providing the preferred name in a target language. A guidance message includes the target language text in the target language.
