(57) Abstract : 
One embodiment of the present invention provides a system for authorizing a software product to be used on a computer system. 
During operation, the system receives authorization information at a database from a point of sale system located at a retailer, wherein 
the authorization information is associated with the software product. The system then modifies a database entry associated with the 
software product to authorize use of the software product.
