(57) Abstract : 
The present invention provides methods and systems for identifying, evaluating, and testing various compounds as memory agents and 
regimens as training protocols related to memory enhancement and/or impairment. Such methods and systems may test any 
compounds or protocols, including the memory agents and training protocols described herein. The methods can comprise combining 
training protocols with the general administration of memory agents. In some embodiments, the present invention may involve 
identifying, selecting, testing, evaluating or assessing a compound as a drug candidate for a memory agent, for example, as one 
effective for enhancing or impairing memory
