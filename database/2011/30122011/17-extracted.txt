(57) Abstract : 
Disclosed is an apparatus and method for controlling packet data transmission between a base station controller (BSC) and a base 
transceiver system (BTS) in a mobile communication network. The BSC transmits as much packet data as an amount that can be 
actually stored in a buffer of the BTS. The BTS has the buffer for temporarily storing the packet data to be transmitted to a mobile 
station, received from the BSC.
