(57) Abstract : 
A device system and method is provided for running mobile applications, including a server for running applications and a client 
installed in a mobile device .The client may connect to the server via a network. The client may display content, receive user input and 
send user input to the server. The server may use the client to receive user input and to display content for running applications for the 
mobile device.
