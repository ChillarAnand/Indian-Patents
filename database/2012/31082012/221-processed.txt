Abstract  =>  Certain aspects of the present disclosure present techniques for requesting and granting bandwidth reservations during network entry and network reentry processes. The subscriber station transmits a request message to the base station including a bandwidth reservation for uplink allocation for at least one subsequent request messages to be transmitted. The base station transmits a response message and an uplink allocation for the at least one subsequent request message to the subscriber station.
