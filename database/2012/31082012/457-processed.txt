Abstract  =>  A gripper type feeding apparatus for the intermittent feeding of a workpiece. The apparatus includes a first linearly guided gripper mechanism which is movable in a first direction of workpiece feeding and in a second direction opposite to the first direction. The first gripper mechanism includes a first gripping member and a second gripping member wherein the second gripping member is movable relative to the first gripping member for gripping the workpiece. The apparatus includes a first release actuator for moving the second gripping member of the first gripper mechanism in a direction relative to the first gripping member of the first gripper mechanism. The apparatus includes a first release connecting link with a first end pivotally connected at a first pivot axis to the first release actuator and with a second end pivotally connected at a second pivot axis to the second gripping member of the first gripper mechanism.
