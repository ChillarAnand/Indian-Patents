(57) Abstract : 
The present invention disclosed a resource configuration method for uplink control channel, The method comprises: network side 
equipment determines downlink carrier that exists cell specific linkage in uplink carrier set and downlink carrier set of User 
Equipment UE; said network side equipment configures uplink control channel resource for downlink carrier which doesnt exist cell 
specific linkage according to said determinate result. The method of the present invention is simple and easy to implement, it could 
use the method in FDD and TDD system to advance system capability of Long Term Evolution multi-carrier update system, and 
advancing compatibility with the present LTE system primely.
