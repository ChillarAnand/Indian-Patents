(57) Abstract : 
Tool for checking a bolt (2) fixed in a surrounding structure (3), comprising a coupling element (7) detachably connectible with the 
bolt (2) in force-transmitting manner ,an engagement section (1a, 1a) which is detachably connectible with the coupling element (7) in 
force-transmitting manner, a rocker (1b, 1b) connected with the engagement section (1a, 1a1) and supportable at the surrounding 
structure (3) to be pivotable and an actuating element (1e) by way of which a force and/or a torque is so introducible into the rocker 
(1b, 1b) that the engagement section (1a, 1a1) can be displaced
