Abstract  =>  The present invention is directed to a tag and labeling system that is used in connection with the disposal of pharmaceutical waste materials that are created in a health care treatment facility. The tag uses a RFID device with an alterable parameter in order to selectively actuate a disposal system to receive a particular type of waste material.
