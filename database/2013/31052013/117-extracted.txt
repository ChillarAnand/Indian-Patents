(57) Abstract :
A radio base station according to the present invention comprising the radio base station eNB#2 comprising a transmission unit 11
configured to transmit C-RNT1 that can be used in the cell #2, to the radio base station eNB #1, in response to HO Preparation (a
handover preparation signal) received from the radio base station eNB #1 that manages the cell #1, and a resource assignment unit
14 configured to notify a radio resource assigned to the mobile station UE by using C-RNTI included in HO Complete, when it is
determined that C-RNTI included in HO Complete received from the mobile station UE that can be used in the cell #2.
