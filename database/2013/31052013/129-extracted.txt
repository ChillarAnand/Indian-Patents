(57) Abstract :
A method for enabling services using a network component in a communication system including a user equipment (UE) is
provided. The UE is configured to require a first message which may be a SIP request. The network component is configured to
create a second message or SIP request based upon the first message. The network component further configured to subsequently
receive a SIP response and select a subsequent action upon receiving the SIP response.
